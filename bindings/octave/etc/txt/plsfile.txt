  plsfile(file)
  *************

  plsfile

  Sets the current output file handle, if applicable. If the file has has
  not been previously opened and is required by the driver, the user will be
  prompted for the file name. This routine, if used, must be called before
  initializing PLplot.

  file (FILE *, input): File pointer.

  NOTE: cannot be called from Fortran.
