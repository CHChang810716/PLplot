Debian Packages
PLplot packages for the Debian GNU/Linux testing distribution (a.k.a sarge, as in March 2003) are available here as a convenience for the users. To upgrade your old packages with APT, include the following lines in your /etc/apt/sources.list file:

  deb http://plplot.sourceforge.net/resources/debian ./
  deb-src http://plplot.sourceforge.net/resources/debian ./

Christian T. Steigies is kindly providing versions of the packages backported to the latest stable distribution of Debian (3.0, a.k.a. woody). The apt-getable repository is accessible through:

  deb http://people.debian.org/~cts/ dists/woody/main/binary-$(ARCH)/   deb http://people.debian.org/~cts/ dists/woody/main/binary-all/   deb-src http://people.debian.org/~cts/ woody main

Debian users running the unstable distribution (a.k.a sid) should point APT to any Debian mirror.
