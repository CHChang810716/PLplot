Labelling the Graph

The function pllab may be called after plenv to write labels on the x and y axes, and at the top of the picture. All the variables are character variables or constants. Trailing spaces are removed and the label is centered in the appropriate field. The syntax for pllab is:

pllab(xlbl, ylbl, toplbl)

xlbl (char *, input) Pointer to string with label for the X-axis (bottom of graph).

ylbl (char *, input) Pointer to string with label for the Y-axis (left of graph).

toplbl (char *, input) Pointer to string with label for the plot (top of picture).

More complex labels can be drawn using the function plmtex. For discussion of writing text in a plot Text, and for more detailed discussion about label generation Annotation.
